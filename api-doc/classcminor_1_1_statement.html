详细描述

用来刻画语句的抽象类

我们支持五种语句：

  • VariableAssignStatement：对一个局部变量赋值
  • SubscriptAssignStatement：对数组中的一个元素赋值
  • FunctionCallStatement：函数调用
  • AssertStatement：包括在 CMinor 中显式写出的 assertion 和 runtime assertion
  • AssumeStatement：只能出现在块首，表示进入这个块需要满足的条件，即控制流结构（分支及循环）的守卫条件
